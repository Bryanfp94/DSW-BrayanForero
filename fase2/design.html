Propósito del sitio

El sitio tiene como proposito, documentar los conceptos respectivos al módulo de caja flexible "Flexbox", sus características y herramientas, indispensables para un diseño adaptable para páginas web.

Conceptos Básicos de Flexbox

El módulo flexbox, generalmente llamado flexbox, está diseñado como un modelo de diseño unidimensional, que es un método que puede ayudar a asignar espacio entre elementos en la interfaz y mejorar las capacidades de alineación. Este artículo revisa las características principales de flexbox, que exploraremos con más detalle en el resto de estas guías. Cuando describimos flexbox como unidimensional, enfatizamos el hecho de que flexbox solo se ocupa del diseño unidimensional a la vez, ya sea como una fila o como una columna. Esto contrasta con el modelo bidimensional de CSS Grid Layout, que maneja tanto columnas como filas. (Conceptos Básicos de Flexbox - CSS | MDN, 2021)
